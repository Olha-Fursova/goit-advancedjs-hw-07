# OOP and TypeScript

The purpose of this work is to practice knowledge on the topic of OOP Principles in TypeScript and consolidate the acquired skills based on the following tasks.

Task 1 - write a short initialization of the class.

Task 2 - create classes with certain properties and settings.

Task 3 - define the game interfaces so that they meet the requirements of the class.

Task 4 - implement a life scenario where a person, a key and a house interact with each other.

Each of these tasks is part of the Advanced JavaScript course with separately allocated hours for passing the TypeScript base with the possibility of applying skills in future practice.
